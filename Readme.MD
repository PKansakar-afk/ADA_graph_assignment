# Setup

## 1. Clone or download the repository

```bash
git clone https://github.com/PKansakar-afk/ADA_graph_assignment.git
cd ADA_graph_assignment
```

## How to Run

All commands should be run from the **root `ADA_graph_assignment/` directory**.

### Run all of Part B (BFS, DFS, Dijkstra, Bellman-Ford, Floyd-Warshall)

```bash
python src/6b/run_part_b.py
```

### Run all of Part C (Prim's MST + greedy discussion)

```bash
python src/7c/run_part_c.py
```

### Run all of Part D (Greedy scheduling + DP scheduling + greedy failure)

```bash
python src/8d/run_part_d.py
```
